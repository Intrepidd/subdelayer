# Subdelayer
A small ruby utility to offset a .srt subtitle file

## Installation

```
gem install 'subdelayer'
```

## Usage

```
# subdelayer offset file.srt

subdelayer 2.4 input.srt > output.srt
subdelayer -4.6 input.srt > output.srt
```
